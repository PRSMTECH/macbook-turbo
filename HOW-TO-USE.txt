CPU MONITOR - HOW TO USE
====================================

EASY LAUNCH OPTIONS (Double-Click from Finder):
------------------------------------------------

1. CPU-CONTROL-CENTER.command
   - Main control panel with all options
   - Double-click to open the control center
   - Choose options using number keys

2. START-CPU-MONITOR.command
   - Launches the menu bar app
   - Shows CPU % in your menu bar
   - Click the percentage for quick actions

3. RUN-CPU-CLEANUP.command
   - Manually run CPU cleanup
   - Kills high-CPU processes
   - Protects your dev tools

4. CHECK-STATUS.command
   - Check protection status
   - See which processes are protected
   - View current CPU usage

FROM TERMINAL:
--------------
cd /Users/bigswizz/cpu-monitor

Quick commands:
./cpu-status      # Check protection status
./cpu-clean       # Run cleanup
./cpu-test        # Test protection

---------------------
Runs every 3 minutes automatically
To disable: launchctl unload ~/Library/LaunchAgents/com.user.cpumanager.plist
To enable: launchctl load ~/Library/LaunchAgents/com.user.cpumanager.plist
